## Write Basic Git Commands Below

- Clone a Repository:
```
$ git clone *url*   under url you actually enter the url of the repo you want to clone
```

- Check Status:
```
$ git status
```

- Stage (Add):
```
$ git add -A
```

- Commit:
```
$ git commit -m "commit message"
```
